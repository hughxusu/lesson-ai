### NLP通用研究流程

1. 阅读论文，研究最新算法。
2. 评估算法的大概方向，确定方案。
   1. 问题相似度评估。
   2. 情景相似度评估。
   3. 语言是否可切换。
   4. 确定方案。
3. 训练数据收集，清洗数据和预处理。
   1. 数据清洗：数据采样、噪声过滤、数据生成
   2. 数据预处理：特征化和数值化后转化为可训练数据
4. 算法实现、系统设计、参数调优和模型升级。
5. 模型效果评估与部署。

## 信息抽取

### 基本概念

#### 定义

非结构化文档 > 预先定义的模板和格式 > 信息表单或表格

涉及两方面因素：

1. 从待分析的文本集中滴定感兴趣的信息
2. 过滤文本集并以一定的格式输出匹配信息

#### 主要任务

* 命名实体识别
* 指代消解
* 利用领域知识推理，在实体-实体、实体-事件之间建立联系

| 名称                                | 作用                         |
| ----------------------------------- | ---------------------------- |
| 命名实体识别(name entity, NE)       | 查找并且对名字、地点进行分类 |
| 指代消解(co-reference, CO)          | 鉴别文本中的实体间恒等关系   |
| 模板元素构建(template element, TE)  | 给实体添加描述信息(使用CO)   |
| 模板关系构建(template relation, TR) | 在TE实体间找出关系           |
| 情景模板构建(scenario template, ST) | 把TE和TR结果放到合适的情景下 |

#### 评估

信息抽取系统要求：

* 准确性
* 弹性
* 自我修改性
* 通用性
* 可扩展性
* 开放性
